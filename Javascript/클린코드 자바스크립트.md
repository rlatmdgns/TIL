# 클린코드 자바스크립트

## 1.과정 파악

### 1-1. 클린코드에 대해 고민하기

- 타인이 정의한 답을 의심합니다.
- 배움에 열린 태도를 가집니다.
- 직접 생각하고 또 고민합니다.
- 클린 코드가 무엇인지를 자바스크립트를 통해 학습합니다.
- 흔히 알려진 자바스크립트 코드 스타일에 대한 견해를 탐구합니다.

### 1-2. JavaScript 특성 파악하기

- 문법 → 자바
- 문자열, 배열, 정규표현식 → 펄
- 함수 → 오크 , 클로저, 스코프 환경 → 스키마
- 프로토타입 → 셀프
- 이벤트 → 하이퍼토크

### 1-3. 사례를 통해 파악하기

똥 코드인지 아닌지 사례를 통해 파악한다.

```jsx
function hello(){
	return 'hello'
}
```
